## 🚀 Installation

```bash
create virtual environment using the following command
python -m venv env
(active this venv) env\Scripts\activate
git clone https://github.com/shm233/To-Do-Management-System.git
cd To-Do-Management-System/todo_project
pip install -r > requirements.txt
python manage.py runserver
pip
```
